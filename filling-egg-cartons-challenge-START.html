JS ALGO CHALLENGE:

Filling Egg Cartons

Write a JavaScript program that fills egg cartons from any number of eggs.

  • Write a function called fillEggCartons(numEggs)
  • numEggs, an integer, is the total number of eggs.
  • The function starts by assigning random prices to each egg.
  • There are 6 possible prices: 25, 30, 35, 40, 45, 50 (cents).
  • The function makes 12-item arrays (cartons) of random egg prices.
  • Calculate the price of each carton, and store the array and price in an object. Ex:
    { eggs: [25,50,40,35,40,35,45,50,30,25,45,35], price: 455 }
  • Save each object to an array of objects called objArr.
  • Sort objArr by price, in ascending order (cheapest carton first).
  • Return objArr to complete the function.

  • BONUS: Improve Price Parity:
  • Reduce the difference between the min and max priced cartons:
  • Change the price of the first 25-cent egg in the cheapest carton to 51.
  • Change the price of the first 50-cent egg in the most expensive carton to 24.
    • The 51 and 24 are so we can tell which cartons were affected by these changes.
  • Recalculate the prices of the two affected cartons.
  • Sort objArr again by price, ascending.
